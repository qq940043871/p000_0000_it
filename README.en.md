2. **B2B Distributed Project** (2020 - Present)
   - Large-scale distributed B2B project providing data exchange, business collaboration and management services for enterprises
   - Adopted Spring Cloud microservice architecture to implement high-concurrency, high-availability enterprise applications

3. **Video Surveillance Platform** (2019 - 2021)
   - Designed and implemented a video surveillance platform supporting live streaming, video playback, video download, etc.
   - Used WebRTC technology to implement real-time video stream transmission

4. **B2C E-commerce System** (2018 - 2019)
   - Responsible for designing and implementing the core architecture of e-commerce systems, including order system, payment system, inventory system and other modules
   - Used Spring Cloud microservice architecture to achieve high availability and scalability of the system

5. **Blockchain Application Development** (2017 - 2018)
   - Ethereum-based blockchain application development, including smart contract development and DApp construction
   - Used Solidity language to develop smart contracts

6. **Clearing Platform Project** (2015 - 2017)
