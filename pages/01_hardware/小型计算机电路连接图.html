CPU 核心电路

CPU 关键引脚说明
引脚组名称方向说明
CLK时钟输入← 输入接晶振，决定 CPU 运行速度
RESET复位← 输入低电平触发，CPU 重新启动
A[0:31]地址总线→ 输出32根线，寻址 4GB 内存空间
D[0:63]数据总线↔ 双向64根线，读写数据
RD/WR读写控制→ 输出高=读，低=写
INT中断请求← 输入硬件设备发出中断信号
INTA中断应答→ 输出CPU 确认收到中断
VCC/GND电源/地← 输入+3.3V 或 +1.2V 核心电压
CPU 时钟电路连接
晶振 (25MHz) ├── 引脚1 → CPU.CLK ← 时钟信号 ├── 引脚2 → GND ← 接地 └── 旁路电容 22pF → GND ← 滤波稳定 复位电路 (RC复位) VCC ──┬── 10kΩ ──┬── CPU.RESET │ │ 100nF └── 按钮 ── GND │ GND 说明：上电时电容充电，RESET 先低后高，CPU 完成复位

内存 (RAM) 电路

SRAM 静态内存引脚（简单版，适合学习）
引脚名称连接到说明
A[0:14]地址线CPU 地址总线15根，寻址 32KB
D[0:7]数据线CPU 数据总线8根，8位数据
CS̄片选（低有效）地址译码器输出选中该内存芯片
OE̅输出使能CPU.RD读操作时拉低
WE̅写使能CPU.WR写操作时拉低
VCC/GND电源+5V / GNDSRAM 用 5V
内存连接电路
CPU SRAM (如 62256) ──── ────────────── A[0:14] ──────────► A[0:14] (地址线直连) D[0:7] ◄──────────► D[0:7] (数据线双向) RD ──────────► OE̅ (读使能) WR ──────────► WE̅ (写使能) 地址译码器 (74HC138) ──────────────────── CPU.A[15:17] → 译码器输入 译码器输出0 → SRAM.CS̄ (选中内存) 译码器输出1 → ROM.CS̄ (选中ROM) 译码器输出2 → IO.CS̄ (选中IO设备) 说明：地址译码器决定哪个设备被选中，避免冲突
💡 现代 DDR 内存：原理相同，但引脚更多（240+），有时钟同步、差分信号等，通常直接用主板 DIMM 插槽，不需要手动连线。

时钟系统（最关键！）

⚡ 为什么时钟最重要？ 计算机里所有操作都要"踩点"进行，时钟就是那个节拍器。没有时钟，CPU 不知道什么时候执行下一条指令，内存不知道什么时候锁存数据。
完整时钟分配系统
晶振 (25MHz 基准) │ ▼ PLL 锁相环 (倍频/分频) ├── ×100 → 2500MHz → CPU 核心时钟 ├── ×4 → 100MHz → 内存总线时钟 ├── ÷1 → 25MHz → PCIe 参考时钟 └── ÷25 → 1MHz → 定时器/RTC 内存时钟 (DDR4 示例) CLK+/CLK- → 差分时钟，直连内存条 频率: 3200MHz (实际传输速率) PCIe 时钟 REFCLK+/REFCLK- → 差分，100MHz → 硬盘/显卡/网卡 共用 RTC 实时时钟 32.768kHz 晶振 → RTC 芯片 纽扣电池供电，断电也走时
时钟信号连接规则
规则原因
走线要短且等长避免时序偏差（skew）
加旁路电容（100nF）滤除高频噪声
高速时钟用差分信号抗干扰，传输更远
时钟线不能打折阻抗不连续会反射
时钟线旁边铺地屏蔽干扰

完整连接步骤（从零开始）

1
先连电源
ATX 电源 → 主板 24pin 接口。CPU 单独 8pin 供电。硬盘 SATA 电源线。所有 GND 必须共地！
2
连时钟
晶振 → PLL → CPU CLK 引脚。这是最先要确认的，没有时钟 CPU 不工作。
3
连复位电路
RC 复位电路 → CPU RESET 引脚。确保上电时 CPU 能正确初始化。
4
连内存
CPU 地址总线 → 内存地址引脚。CPU 数据总线 ↔ 内存数据引脚。地址译码器决定内存映射范围。
5
连存储（硬盘/ROM）
BIOS ROM → 地址总线（固定地址 0xFFFFFFF0）。硬盘 → SATA 控制器 → PCIe 总线 → CPU。
6
连 I/O 设备
键盘/鼠标 → USB 控制器 → PCIe → CPU。中断线 IRQ → APIC → CPU INT 引脚。
7
验证总线
用示波器检查时钟波形是否正常。用逻辑分析仪检查总线数据。上电测试 POST（开机自检）。
