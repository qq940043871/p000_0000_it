## 一、技术栈概览

### 1.1 核心技术能力

| 领域 | 掌握程度 | 关键技能 |
|------|----------|----------|
| **Java 基础** | ⭐⭐⭐⭐⭐ | 集合源码、JVM原理、并发编程、设计模式 |
| **Spring 全家桶** | ⭐⭐⭐⭐⭐ | Spring Boot、Spring Cloud、Spring Security |
| **数据库** | ⭐⭐⭐⭐ | MySQL、Redis、ElasticSearch、分库分表 |
| **消息队列** | ⭐⭐⭐⭐ | Kafka、RabbitMQ、异步解耦 |
| **容器化** | ⭐⭐⭐⭐ | Docker、K8s、服务网格 |
| **分布式系统** | ⭐⭐⭐⭐ | 一致性、事务、限流熔断 |
| **AI/大模型** | ⭐⭐⭐⭐ | RAG系统、向量数据库、内网部署、微调 |
| **架构设计** | ⭐⭐⭐⭐⭐ | 高可用、高并发、可扩展、安全合规 |

### 1.2 技术栈明细

```
【后端框架】
├── Spring Boot 2.x/3.x（核心原理、启动流程）
├── Spring Cloud Alibaba（注册、配置、熔断、网关）
├── MyBatis-Plus（SQL优化、批量操作）
└── 异步编程（CompletableFuture、线程池调优）

【数据库与缓存】
├── MySQL（索引优化、分库分表、读写分离）
├── Redis（集群、持久化、分布式锁、缓存策略）
├── Elasticsearch（倒排索引、聚合查询、日志分析）
└── ClickHouse（OLAP、大数据聚合）

【消息队列】
├── Kafka（分区、幂等、事务Exactly-once）
├── RabbitMQ（交换机、延迟队列、死信队列）
└── RocketMQ（顺序消息、事务消息）

【DevOps 与运维】
├── Docker（镜像优化、多阶段构建、私有仓库）
├── Kubernetes（Pod、Service、Ingress、Helm）
├── Prometheus + Grafana（监控告警）
└── ELK/Loki（日志采集与分析）

【AI/大模型】
├── 模型部署：vLLM、Ollama（内网私有化）
├── 向量数据库：Milvus、Qdrant（Pinecone对比）
├── RAG框架：LangChain、LlamaIndex
├── Embedding：BGE-M3、text2vec
├── Agent框架：AutoGen
└── 微调工具：LoRA、QLoRA（模型蒸馏）

【前端】
├── Vue3（Composition API、响应式原理）
├── 性能优化（CDN、懒加载、缓存策略）
└── Node.js（SSR、同构架构）

【架构能力】
├── 高并发：线程池、异步、NIO、连接池
├── 高可用：限流、熔断、降级、隔离
├── 可扩展：微服务拆分、服务网格
└── 安全：OAuth2、JWT、数据脱敏、审计日志
```

### 3.1 目标岗位

| 优先级 | 岗位 | 薪资预期 | 城市 |

| P1 | 全栈架构师 / 技术负责人 | 22-28K | 郑州 |

| P3 | 高级Java开发（技术导向） | 20-25K | 郑州 |

### 3.2 核心竞争力

1. **全栈能力**：从前端Vue到后端Spring Cloud到数据库优化，完整技术链路
2. **AI落地经验**：有真实的RAG系统设计和内网部署经验，不是纸上谈兵
3. **电商业务深度**：订单、支付、库存、会员等核心模块的架构设计能力
4. **工程化思维**：注重方案权衡、性能调优、成本控制
