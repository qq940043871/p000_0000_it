# 以太坊-0.7.6合约（源码）

## 智能合约概述

以太坊智能合约是运行在以太坊虚拟机（EVM）上的自动执行代码。本教程将深入分析以太坊智能合约的源码实现，包括合约创建、执行和交互。

## 智能合约结构

### 合约代码

智能合约代码通常使用Solidity语言编写，编译后生成EVM字节码。

### 合约账户

智能合约在以太坊网络上以账户的形式存在，具有以下特点：

- **地址**：由创建交易的发送地址和nonce生成
- **余额**：可以接收和发送以太币
- **代码**：存储智能合约的字节码
- **存储**：存储合约的状态数据

### 执行消息

```go
// executeMessage executes a message.
func (evm *EVM) executeMessage(msg *Message, contractCreation bool) (ret []byte, contractAddr common.Address, leftOverGas uint64, err error) {
    // 检查消息
    if msg.gas == 0 {
        return nil, common.Address{}, 0, errGasLimitReached
    }
    
    // 检查余额
    if evm.StateDB.GetBalance(msg.caller).Cmp(msg.value) < 0 {
        return nil, common.Address{}, msg.gas, errInsufficientBalance
    }
    
    // 执行消息
    if contractCreation {
        // 合约创建
        ret, contractAddr, leftOverGas, err = evm.create(msg)
    } else {
        // 消息调用
        ret, leftOverGas, err = evm.call(msg)
    }
    
    return ret, contractAddr, leftOverGas, err
}
```

### 存储操作

```go
// SLOAD implements the SLOAD instruction.
func opSload(pc *uint64, evm *EVM, contract *Contract, memory *Memory, stack *Stack) ([]byte, error) {
    // 获取存储键
    key := stack.pop()
    
    // 从存储中读取值
    value := evm.StateDB.GetState(contract.Address(), common.Hash(key.Bytes32()))
    
    // 将值压入栈
    stack.push(value.Big())
    
    return nil, nil
}

// SSTORE implements the SSTORE instruction.
func opSstore(pc *uint64, evm *EVM, contract *Contract, memory *Memory, stack *Stack) ([]byte, error) {
    // 获取存储键和值
    value := stack.pop()
    key := stack.pop()
    
    // 计算Gas
    var gas uint64
    current := evm.StateDB.GetState(contract.Address(), common.Hash(key.Bytes32()))
    if current == (common.Hash{}) {
        gas = params.SstoreSetGas
    } else if value.Sign() == 0 {
        gas = params.SstoreClearGas
    } else {
        gas = params.SstoreResetGas
    }
    
    // 检查Gas
    if contract.UseGas(gas) {}
    
    // 写入存储
    evm.StateDB.SetState(contract.Address(), common.Hash(key.Bytes32()), common.Hash(value.Bytes32()))
    
    return nil, nil
}
```

## 合约调用

### 调用类型

以太坊支持多种类型的合约调用：

- **CALL**：普通调用，更改调用者
- **CALLCODE**：代码调用，使用当前合约的存储
- **DELEGATECALL**：委托调用，使用当前合约的存储和调用者
- **STATICCALL**：静态调用，不允许修改状态

### 调用实现

```go
// call executes a message call.
func (evm *EVM) call(msg *Message) (ret []byte, leftOverGas uint64, err error) {
    // 获取接收者地址
    addr := *msg.to
    
    // 检查接收者是否存在
    if !evm.StateDB.Exist(addr) {
        return nil, msg.gas, nil
    }
    
    // 检查接收者是否有代码
    code := evm.StateDB.GetCode(addr)
    if len(code) == 0 {
        return nil, msg.gas, nil
    }
    
    // 创建合约对象
    contract := NewContract(msg.caller, AccountRef(addr), msg.value, msg.gas)
    contract.SetCodeOptionalHash(&addr, code)
    
    // 执行合约代码
    ret, err = evm.interpreter.Run(contract, msg.data)
    
    // 返回结果
    return ret, contract.Gas, err
}
```

```go
func opLog0(pc *uint64, evm *EVM, contract *Contract, memory *Memory, stack *Stack) ([]byte, error) {
    // 获取内存偏移量和长度
    offset := stack.pop()
    length := stack.pop()
    
    // 计算Gas
    gas := params.LogGas + params.LogTopicGas*uint64(0) + params.LogDataGas*uint64(length.Uint64())
    if contract.UseGas(gas) {}
```

## 合约安全

### 常见安全问题

- **重入攻击**：合约在执行过程中被外部调用再次进入
- **整数溢出**：整数运算导致的溢出
- **访问控制**：缺少对敏感操作的访问控制
- **Gas限制**：无限循环导致的Gas耗尽
- **随机数生成**：不安全的随机数生成

### 安全防护

- **重入锁**：使用状态变量防止重入攻击
- **安全数学库**：使用SafeMath等库防止整数溢出
- **访问控制修饰符**：使用onlyOwner等修饰符限制访问
- **Gas限制**：避免无限循环，设置合理的Gas限制
- **链上随机数**：使用链上随机数生成方法
