# 以太坊-0.7.1账户（源码）

## 账户概述

以太坊账户是存储以太币和执行智能合约的基本单位。以太坊中有两种类型的账户：外部账户（EOA）和合约账户。本教程将深入分析以太坊账户的源码实现。

## 账户类型

### 外部账户（EOA）

- **特点**：由私钥控制，没有代码
- **功能**：发送交易、接收以太币
- **结构**：地址、余额、nonce

### 合约账户

- **特点**：由代码控制，有智能合约代码
- **功能**：执行智能合约、存储数据
- **结构**：地址、余额、代码、存储

## 账户源码分析

### 账户数据结构

在以太坊源码中，账户的数据结构定义在`core/state/state_object.go`文件中：

```go
// StateObject represents an Ethereum account which is being modified.
type StateObject struct {
    address  common.Address
    addrHash common.Hash // hash of ethereum address of the account
    data     Account
    db       *StateDB
    trie     Trie // storage trie, which becomes non-nil on first access
    code     Code // contract bytecode, which gets set when code is loaded
    origin   Account // Original state object, used to check if state was modified
    
    // Cache flags
    dirtyCode bool // true if the code was updated
    codeSize  int  // size of the contract code, used when doing size checks
    
    // Mark for deletion.置为true时，账户会在状态提交时被删除
    suicide bool
    
    // Mark for resetting storage.置为true时，存储会在状态提交时被重置
    resetStorage bool
    
    // Precompiled contract flag
    precompile bool
}

// Account is the Ethereum consensus representation of accounts.
type Account struct {
    Nonce    uint64
    Balance  *big.Int
    Root     common.Hash // merkle root of the storage trie
    CodeHash common.Hash
}
```

### 账户操作

#### 创建账户

```go
// createObject creates a new state object.
func (db *StateDB) createObject(addr common.Address) (newobj *StateObject) {
    newobj = &StateObject{
        db:       db,
        address:  addr,
        addrHash: crypto.Keccak256Hash(addr.Bytes()),
        data: Account{
            Nonce:   0,
            Balance: new(big.Int),
            Root:    emptyRoot,
        },
        origin: Account{
            Nonce:   0,
            Balance: new(big.Int),
            Root:    emptyRoot,
        },
    }
    if db.db == nil {
        newobj.trie, _ = db.trieProvider(addrHash, common.Hash{}) // Create a temporary trie
    } else {
        newobj.trie, _ = db.trieProvider(addrHash, newobj.data.Root) // Create a trie with the account's storage root
    }
    db.journal.append(createObjectChange{account: newobj})
    db.stateObjects[addr] = newobj
    return newobj
}
```

```go
// GetOrNewStateObject retrieves a state object or creates a new state object if it does not exist
func (db *StateDB) GetOrNewStateObject(addr common.Address) *StateObject {
    if obj, found := db.stateObjects[addr]; found {
        if obj.suicide {
            return nil
        }
        return obj
    }
    obj := db.createObject(addr)
    return obj
}
```

## 账户存储

### 存储结构

账户的存储使用Merkle Patricia Trie实现，定义在`core/state/state_object.go`文件中：

```go
// GetStorage returns the value of key in the account storage
func (s *StateObject) GetStorage(key common.Hash) common.Hash {
    // If we're accessing the storage of a self-destructed account, return nothing
    if s.suicide {
        return common.Hash{}
    }
    // If the trie is not yet loaded, load it
    if s.trie == nil {
        s.trie, _ = s.db.trieProvider(s.addrHash, s.data.Root)
    }
    // Get the value from the trie
    value, err := s.trie.TryGet(key.Bytes())
    if err != nil {
        s.db.setError(err)
        return common.Hash{}
    }
    return common.BytesToHash(value)
}

// SetStorage sets the value of key in the account storage
func (s *StateObject) SetStorage(key, value common.Hash) {
    // If we're accessing the storage of a self-destructed account, don't do anything
    if s.suicide {
        return
    }
    // If the trie is not yet loaded, load it
    if s.trie == nil {
        s.trie, _ = s.db.trieProvider(s.addrHash, s.data.Root)
    }
    // Set the value in the trie
    s.db.journal.append(storageChange{account: s, key: key, prev: s.GetStorage(key)})
    err := s.trie.TryUpdate(key.Bytes(), value.Bytes())
    if err != nil {
        s.db.setError(err)
    }
}
```

## 智能合约代码

### 代码存储

```go
// SetCode sets the contract code
func (s *StateObject) SetCode(codeHash common.Hash, code []byte) {
    prevcode := s.Code()
    s.db.journal.append(codeChange{account: s, prevhash: s.data.CodeHash, prevcode: prevcode})
    s.data.CodeHash = codeHash
    s.code = code
    s.dirtyCode = true
    s.codeSize = len(code)
}

// Code returns the contract code
func (s *StateObject) Code() []byte {
    if s.code != nil {
        return s.code
    }
    if s.data.CodeHash == emptyCodeHash {
        return nil
    }
    code, err := s.db.db.Get(s.data.CodeHash.Bytes())
    if err != nil {
        s.db.setError(fmt.Errorf("code %x: %v", s.data.CodeHash, err))
    }
    s.code = code
    s.codeSize = len(code)
    return code
}
```

## 账户状态管理

### 状态提交

```go
// Commit commits the state changes to the underlying trie
func (db *StateDB) Commit(deleteEmptyObjects bool) (root common.Hash, err error) {
    // Commit all state objects
    for addr, stateObject := range db.stateObjects {
        if stateObject.suicide {
            db.deleteStateObject(stateObject)
        } else if stateObject.dirtyCode || deleteEmptyObjects && stateObject.empty() {
            db.deleteStateObject(stateObject)
        } else if stateObject.dirty {
            // Write the account to the trie
            stateObject.commitTrie()
            db.updateStateObject(stateObject)
        }
    }
    // Commit the main trie
    return db.trie.Commit(func(leaf []byte, parent common.Hash) error {
        var account Account
        if err := rlp.DecodeBytes(leaf, &account); err != nil {
            return err
        }
        if account.Root != emptyRoot {
            db.db.TrieDB().Reference(account.Root, parent)
        }
        if account.CodeHash != emptyCodeHash {
            db.db.TrieDB().Reference(account.CodeHash, parent)
        }
        return nil
    })
}
```

## 账户地址生成

### 地址生成算法

以太坊账户地址由公钥生成，使用Keccak-256哈希算法：

```go
// PubkeyToAddress returns the address of the given public key
func PubkeyToAddress(p ecdsa.PublicKey) common.Address {
    pubBytes := FromECDSAPub(&p)
    return common.BytesToAddress(crypto.Keccak256(pubBytes[1:])[12:])
}
```
