# 以太坊-0.7.2区块、区块链

## 区块概述

区块是以太坊区块链的基本单位，包含交易数据、状态变更和其他元数据。本教程将介绍以太坊区块的结构、生成过程和验证机制。

## 区块结构

### 区块头结构

在以太坊源码中，区块头的数据结构定义在`core/types/block.go`文件中：

```go
// Header represents a block header in the Ethereum blockchain.
type Header struct {
    ParentHash  common.Hash    `json:"parentHash"       gencodec:"required"`
    UncleHash   common.Hash    `json:"sha3Uncles"       gencodec:"required"`
    Coinbase    common.Address `json:"miner"            gencodec:"required"`
    Root        common.Hash    `json:"stateRoot"        gencodec:"required"`
    TxHash      common.Hash    `json:"transactionsRoot" gencodec:"required"`
    ReceiptHash common.Hash    `json:"receiptsRoot"     gencodec:"required"`
    Bloom       Bloom          `json:"logsBloom"        gencodec:"required"`
    Difficulty  *big.Int       `json:"difficulty"       gencodec:"required"`
    Number      *big.Int       `json:"number"           gencodec:"required"`
    GasLimit    uint64         `json:"gasLimit"         gencodec:"required"`
    GasUsed     uint64         `json:"gasUsed"          gencodec:"required"`
    Time        uint64         `json:"timestamp"        gencodec:"required"`
    Extra       []byte         `json:"extraData"        gencodec:"required"`
    MixDigest   common.Hash    `json:"mixHash"          gencodec:"required"`
    Nonce       BlockNonce     `json:"nonce"            gencodec:"required"`
}
```

### 区块头字段说明

- **ParentHash**：前一个区块的哈希值
- **UncleHash**：叔区块的哈希值
- **Coinbase**：矿工地址
- **Root**：状态根，Merkle Patricia Trie的根哈希
- **TxHash**：交易根，交易Merkle树的根哈希
- **ReceiptHash**：收据根，收据Merkle树的根哈希
- **Bloom**：日志 bloom 过滤器
- **Difficulty**：挖矿难度
- **Number**：区块号
- **GasLimit**：区块的Gas上限
- **GasUsed**：区块中所有交易使用的Gas总和
- **Time**：区块时间戳
- **Extra**：额外数据
- **MixDigest**：混合哈希，用于工作量证明
- **Nonce**：随机数，用于工作量证明

## 区块生成

### 区块创建

```go
// NewBlock creates a new block.
func NewBlock(header *Header, txs []*Transaction, uncles []*Header, receipts []*Receipt) *Block {
    b := &Block{
        header:       CopyHeader(header),
        transactions: make(Transactions, len(txs)),
        uncles:       make([]*Header, len(uncles)),
    }
    copy(b.transactions, txs)
    for i, uncle := range uncles {
        b.uncles[i] = CopyHeader(uncle)
    }
    return b
}
```

### 区块挖矿

区块挖矿的过程包括：

1. **准备区块头**：设置区块头的各个字段
2. **工作量证明**：通过调整nonce值，使区块头的哈希值满足难度要求
3. **验证区块**：验证区块的有效性
4. **广播区块**：将区块广播到网络

## 区块验证

### 区块验证过程

1. **验证区块头**：检查区块头的各个字段是否有效
2. **验证工作量证明**：检查区块的哈希值是否满足难度要求
3. **验证交易**：验证区块中的所有交易
4. **验证状态**：执行区块中的所有交易，验证状态变更
5. **验证叔区块**：验证叔区块的有效性

### 区块验证代码

在`core/block_validator.go`文件中，区块验证的逻辑如下：

```go
// ValidateBlock validates the given block.
func (v *BlockValidator) ValidateBlock(block *types.Block) error {
    // Check that the block is not nil
    if block == nil {
        return ErrNilBlock
    }
    // Check that the block header is valid
    if err := v.ValidateHeader(block.Header(), true); err != nil {
        return err
    }
    // Check that the block body is valid
    if err := v.ValidateBody(block); err != nil {
        return err
    }
    return nil
}

// ValidateHeader validates the given block header.
func (v *BlockValidator) ValidateHeader(header *types.Header, seal bool) error {
    // Check that the header is not nil
    if header == nil {
        return ErrNilHeader
    }
    // Check that the header has a valid parent
    parent := v.bc.GetBlockByHash(header.ParentHash)
    if parent == nil {
        return ErrUnknownAncestor
    }
    // Check that the header number is correct
    if header.Number.Cmp(new(big.Int).Add(parent.Number(), big.NewInt(1))) != 0 {
        return ErrInvalidNumber
    }
    // Check that the header time is correct
    if header.Time <= parent.Header().Time {
        return ErrInvalidTimestamp
    }
    // Check that the header gas limit is correct
    if err := ValidateGasLimit(parent.Header().GasLimit, header.GasLimit); err != nil {
        return err
    }
    // Check that the header difficulty is correct
    expectedDifficulty := CalcDifficulty(v.config, header.Time, parent.Header())
    if header.Difficulty.Cmp(expectedDifficulty) != 0 {
        return ErrInvalidDifficulty
    }
    // Check that the header has a valid seal
    if seal {
        if err := v.verifySeal(header); err != nil {
            return err
        }
    }
    return nil
}
```

## 区块链结构

### 区块链存储

以太坊区块链使用LevelDB存储区块数据，区块按哈希值索引。

### 区块链同步

区块链同步的过程包括：

1. **发现对等节点**：通过P2P网络发现其他节点
2. **获取区块头**：从对等节点获取区块头
3. **验证区块头**：验证区块头的有效性
4. **获取区块体**：从对等节点获取区块体
5. **验证区块体**：验证区块体的有效性
6. **更新本地区块链**：将验证通过的区块添加到本地区块链

### 区块链分叉

当网络中同时出现两个不同的区块时，会产生分叉。以太坊使用最长链规则来解决分叉：

1. **选择最长链**：选择包含最多工作量的链
2. **重新组织链**：如果发现更长的链，重新组织本地区块链

```go
// GetBlockByHash retrieves a block by its hash.
func (bc *BlockChain) GetBlockByHash(hash common.Hash) *types.Block {
    block := bc.db.GetBlock(hash, bc.GetBlockNumber(hash))
    if block == nil {
        return nil
    }
    return block
}

// GetBlockByNumber retrieves a block by its number.
func (bc *BlockChain) GetBlockByNumber(number *big.Int) *types.Block {
    hash := bc.GetBlockHash(number)
    if hash == (common.Hash{}) {
        return nil
    }
    return bc.GetBlockByHash(hash)
}
```

```go
func (bc *BlockChain) GetTransaction(hash common.Hash) (*types.Transaction, common.Hash, uint64, uint64) {
    tx, blockHash, blockNumber, index := bc.txLookup[hash]
    if tx == nil {
        return nil, common.Hash{}, 0, 0
    }
    return tx, blockHash, blockNumber, index
}
```

## 区块奖励

### 区块奖励计算

```go
// BlockReward returns the block reward for the given block.
func BlockReward(blockNumber *big.Int) *big.Int {
    reward := new(big.Int).Set(blockReward)
    // Apply reward reduction after 5 million blocks
    if blockNumber.Cmp(big.NewInt(5000000)) >= 0 {
        // Reduce reward by half every 5 million blocks
        for i := big.NewInt(5000000); blockNumber.Cmp(i) >= 0; i.Add(i, big.NewInt(5000000)) {
            reward.Div(reward, big.NewInt(2))
        }
    }
    return reward
}
```

### 叔区块奖励

```go
// UncleReward returns the uncle reward for the given block and uncle.
func UncleReward(blockNumber, uncleNumber *big.Int) *big.Int {
    baseReward := BlockReward(blockNumber)
    // Uncle reward is (8 - (blockNumber - uncleNumber)) / 8 * baseReward
    diff := new(big.Int).Sub(blockNumber, uncleNumber)
    numerator := new(big.Int).Sub(big.NewInt(8), diff)
    reward := new(big.Int).Mul(baseReward, numerator)
    reward.Div(reward, big.NewInt(8))
    return reward
}
```
