# 以太坊-0.7.5存储（源码）

## 存储概述

以太坊存储是以太坊区块链的核心组成部分，它负责存储账户状态、交易数据和智能合约代码。本教程将深入分析以太坊存储的源码实现，包括存储结构、存储操作和存储优化。

## 存储结构

### 状态树

以太坊使用Merkle Patricia Trie作为状态树，定义在`trie`目录中。

```go
// Trie is a Merkle Patricia Trie.
type Trie struct {
    root         node
    db           Database
    originalRoot common.Hash
}

// node is the interface for nodes in the trie.
type node interface {
    fstring(string) string
    cache() (hashNode, bool)
    copy() node
    rlp() []byte
    len() int
    hash() hashNode
}

// Database is the interface for trie databases.
type Database interface {
    Get(key []byte) (value []byte, err error)
    Put(key []byte, value []byte) error
    Delete(key []byte) error
    Commit() error
    Close() error
}
```

### 存储类型

以太坊的存储包括以下几种类型：

1. **状态存储**：存储账户状态
2. **交易存储**：存储交易数据
3. **收据存储**：存储交易执行结果
4. **代码存储**：存储智能合约代码

```go
type Account struct {
    Nonce    uint64
    Balance  *big.Int
    Root     common.Hash // merkle root of the storage trie
    CodeHash common.Hash
}

// StateObject represents an Ethereum account which is being modified.
type StateObject struct {
    address  common.Address
    addrHash common.Hash // hash of ethereum address of the account
    data     Account
    db       *StateDB
    trie     Trie // storage trie, which becomes non-nil on first access
    code     Code // contract bytecode, which gets set when code is loaded
    origin   Account // Original state object, used to check if state was modified
    
    // Cache flags
    dirtyCode bool // true if the code was updated
    codeSize  int  // size of the contract code, used when doing size checks
    
    // Mark for deletion.置为true时，账户会在状态提交时被删除
    suicide bool
    
    // Mark for resetting storage.置为true时，存储会在状态提交时被重置
    resetStorage bool
    
    // Precompiled contract flag
    precompile bool
}
```

### 状态操作

```go
// GetState retrieves a value from the account storage.
func (s *StateObject) GetState(db Database, key common.Hash) common.Hash {
    // If we're accessing the storage of a self-destructed account, return nothing
    if s.suicide {
        return common.Hash{}
    }
    // If the trie is not yet loaded, load it
    if s.trie == nil {
        s.trie, _ = db.OpenStorageTrie(s.addrHash, s.data.Root)
    }
    // Get the value from the trie
    value, err := s.trie.TryGet(key.Bytes())
    if err != nil {
        s.db.setError(err)
        return common.Hash{}
    }
    return common.BytesToHash(value)
}

// SetState sets a value in the account storage.
func (s *StateObject) SetState(db Database, key, value common.Hash) {
    // If we're accessing the storage of a self-destructed account, don't do anything
    if s.suicide {
        return
    }
    // If the trie is not yet loaded, load it
    if s.trie == nil {
        s.trie, _ = db.OpenStorageTrie(s.addrHash, s.data.Root)
    }
    // Set the value in the trie
    s.db.journal.append(storageChange{account: s, key: key, prev: s.GetState(db, key)})
    err := s.trie.TryUpdate(key.Bytes(), value.Bytes())
    if err != nil {
        s.db.setError(err)
    }
}
```

```go
// SetCode sets the contract code.
func (s *StateObject) SetCode(codeHash common.Hash, code []byte) {
    prevcode := s.Code()
    s.db.journal.append(codeChange{account: s, prevhash: s.data.CodeHash, prevcode: prevcode})
    s.data.CodeHash = codeHash
    s.code = code
    s.dirtyCode = true
    s.codeSize = len(code)
}

// Code returns the contract code.
func (s *StateObject) Code() []byte {
    if s.code != nil {
        return s.code
    }
    if s.data.CodeHash == emptyCodeHash {
        return nil
    }
    code, err := s.db.db.Get(s.data.CodeHash.Bytes())
    if err != nil {
        s.db.setError(fmt.Errorf("code %x: %v", s.data.CodeHash, err))
    }
    s.code = code
    s.codeSize = len(code)
    return code
}
```

## 存储操作

### 状态提交

```go
// Commit commits the state changes to the underlying trie.
func (db *StateDB) Commit(deleteEmptyObjects bool) (root common.Hash, err error) {
    // Commit all state objects
    for addr, stateObject := range db.stateObjects {
        if stateObject.suicide {
            db.deleteStateObject(stateObject)
        } else if stateObject.dirtyCode || deleteEmptyObjects && stateObject.empty() {
            db.deleteStateObject(stateObject)
        } else if stateObject.dirty {
            // Write the account to the trie
            stateObject.commitTrie()
            db.updateStateObject(stateObject)
        }
    }
    // Commit the main trie
    return db.trie.Commit(func(leaf []byte, parent common.Hash) error {
        var account Account
        if err := rlp.DecodeBytes(leaf, &account); err != nil {
            return err
        }
        if account.Root != emptyRoot {
            db.db.TrieDB().Reference(account.Root, parent)
        }
        if account.CodeHash != emptyCodeHash {
            db.db.TrieDB().Reference(account.CodeHash, parent)
        }
        return nil
    })
}
```

### 状态回滚

```go
// Revert reverts the state changes to the last checkpoint.
func (db *StateDB) RevertToSnapshot(snapshot int) {
    // Revert the journal
    db.journal.revert(snapshot)
    
    // Clear the state objects
    for addr := range db.stateObjects {
        delete(db.stateObjects, addr)
    }
    
    // Rebuild the state objects
    db.stateObjects = make(map[common.Address]*StateObject)
}
```

## 存储优化

### 缓存

以太坊使用多种缓存来优化存储操作：

- **状态缓存**：缓存最近访问的账户状态
- **Trie缓存**：缓存最近访问的trie节点
- **代码缓存**：缓存最近访问的智能合约代码

### 数据库优化

以太坊使用LevelDB作为底层数据库，通过以下方式优化存储：

- **批量写入**：减少数据库写入次数
- **压缩**：压缩存储数据
- **前缀编码**：优化键的存储

## 存储安全

### 数据完整性

以太坊通过Merkle树确保数据的完整性：

- **状态根**：状态树的根哈希
- **交易根**：交易树的根哈希
- **收据根**：收据树的根哈希

### 防篡改

以太坊的存储结构使得数据一旦写入，难以篡改：

- **Merkle Patricia Trie**：任何数据修改都会改变根哈希
- **工作量证明**：修改数据需要重新计算工作量证明
