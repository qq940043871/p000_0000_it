## 🚦 二、流量控制

### 2.1 限流算法

```java
/**
 * 令牌桶限流算法
 */
public class TokenBucketRateLimiter {
    private final double rate;        // 令牌生成速率
    private final double capacity;   // 桶容量
    private double tokens;            // 当前令牌数
    private long lastRefillTime;      // 上次填充时间
    
    public boolean tryAcquire(int permits) {
        refillTokens();
        
        if (tokens >= permits) {
            tokens -= permits;
            return true;
        }
        return false;
    }
    
    private void refillTokens() {
        long now = System.currentTimeMillis();
        double elapsed = (now - lastRefillTime) / 1000.0;
        tokens = Math.min(capacity, tokens + elapsed * rate);
        lastRefillTime = now;
    }
}
```

## 💥 三、熔断降级

### 3.1 Sentinel 熔断策略

```java
@RestController
public class OrderController {
    
    @GetMapping("/order/{id}")
    @SentinelResource(value = "getOrder",
                     blockHandler = "getOrderBlock",
                     fallback = "getOrderFallback")
    public Result<Order> getOrder(@PathVariable Long id) {
        return orderService.getById(id);
    }
    
    // 限流/熔断处理
    public Result<Order> getOrderBlock(Long id, BlockException e) {
        return Result.fail("系统繁忙，请稍后再试");
    }
    
    // 异常降级处理
    public Result<Order> getOrderFallback(Long id, Throwable t) {
        log.error("查询订单异常", t);
        return Result.fail("服务暂时不可用");
    }
}
```

### 3.2 降级策略

| 等级 | 措施 | 影响 |
|------|------|------|
| **L1** | 关闭非核心功能 | 部分功能不可用 |
| **L2** | 简化流程 | 减少校验步骤 |
| **L3** | 返回缓存数据 | 数据可能过期 |
| **L4** | 返回默认值 | 功能不可用 |

## 📦 四、消息队列解耦

### 4.1 异步化架构

```
同步模式：
用户下单 ──► 创建订单 ──► 扣减库存 ──► 扣减余额 ──► 发送通知
                │           │           │            │
                ▼           ▼           ▼            ▼
              50ms        30ms        20ms         10ms
              ─────────────────────────────
                   总耗时: 110ms

异步模式：
用户下单 ──► 创建订单(50ms) ──► 返回成功
                │
                ▼
            MQ消息队列
                │
       ┌────────┼────────┐
       │        │        │
       ▼        ▼        ▼
    扣减库存  扣减余额  发送通知
    (异步)    (异步)    (异步)
```

### 4.2 Kafka 消息处理

```java
@KafkaListener(topics = "order-created")
public void handleOrderCreated(OrderCreatedEvent event) {
    try {
        // 1. 扣减库存
        inventoryService.deduct(event.getSkuId(), event.getCount());
        
        // 2. 发送库存预警
        if (inventoryService.isLowStock(event.getSkuId())) {
            alertService.sendLowStockAlert(event.getSkuId());
        }
        
        // 3. 记录日志
        log.info("订单创建后续处理完成: orderId={}", event.getOrderId());
        
    } catch (Exception e) {
        // 消息重试
        throw new KafkaRetryException("处理失败，等待重试", e);
    }
}
```

### 5.2 Redis 库存预扣

```lua
-- Redis 库存扣减脚本
local stock = redis.call('GET', KEYS[1])

if not stock then
    return -1  -- 商品不存在
end

local stockNum = tonumber(stock)
local quantity = tonumber(ARGV[1])

if stockNum < quantity then
    return 0  -- 库存不足
end

-- 扣减库存
redis.call('DECRBY', KEYS[1], quantity)

-- 返回剩余库存
return stockNum - quantity
```

### 6.2 本地缓存实现

```java
@Configuration
public class CaffeineConfig {
    
    @Bean
    public Cache<String, Object> localCache() {
        return Caffeine.newBuilder()
            .maximumSize(10000)           // 最大条目数
            .expireAfterWrite(1, TimeUnit.MINUTES)  // 写后过期
            .recordStats()                 // 记录统计
            .build();
    }
}

// 使用示例
@Service
public class ProductService {
    
    @Autowired
    private Cache<String, Object> localCache;
    
    @Autowired
    private RedisTemplate redisTemplate;
    
    public Product getProduct(Long id) {
        String cacheKey = "product:" + id;
        
        // 1. 本地缓存
        Product product = localCache.getIfPresent(cacheKey);
        if (product != null) {
            return product;
        }
        
        // 2. Redis缓存
        product = (Product) redisTemplate.opsForValue().get(cacheKey);
        if (product != null) {
            localCache.put(cacheKey, product);
            return product;
        }
        
        // 3. 数据库
        product = productDao.selectById(id);
        
        // 4. 写入缓存
        if (product != null) {
            redisTemplate.opsForValue().set(cacheKey, product, 10, TimeUnit.MINUTES);
            localCache.put(cacheKey, product);
        }
        
        return product;
    }
}
```
