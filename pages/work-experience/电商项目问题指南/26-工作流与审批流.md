## 📖 概述

工作流是B2B系统的核心功能。本章介绍工作流引擎设计、审批流程配置、动态表单集成。

## 🏗️ 一、工作流引擎架构

### 1.1 核心概念

```
工作流三要素：

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  流程定义 (Process Definition)                           │
│  ├── 流程ID、名称、版本                                  │
│  ├── 节点定义 (Nodes)                                   │
│  │   ├── 开始节点 (Start)                               │
│  │   ├── 用户任务节点 (UserTask)                        │
│  │   ├── 服务任务节点 (ServiceTask)                     │
│  │   ├── 排他网关 (ExclusiveGateway)                    │
│  │   ├── 并行网关 (ParallelGateway)                    │
│  │   ├── 包容网关 (InclusiveGateway)                   │
│  │   └── 结束节点 (End)                                 │
│  └── 连线定义 (Sequences)                              │
│      └── 条件表达式                                     │
│                                                          │
│  流程实例 (Process Instance)                            │
│  ├── 业务关联 (BusinessKey)                            │
│  ├── 当前节点                                          │
│  ├── 历史轨迹                                          │
│  └── 变量 (Variables)                                  │
│                                                          │
│  任务实例 (Task Instance)                               │
│  ├── 任务类型                                          │
│  ├── 办理人/候选人                                      │
│  ├── 任务状态                                          │
│  └── 任务变量                                          │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 1.2 工作流引擎对比

| 引擎 | 特点 | 适用场景 | 国产适配 |
|------|------|----------|----------|
| **Flowable** | 功能完善、性能好 | 企业级 | 良好 |
| **Camunda** | BPMN 2.0标准 | 复杂流程 | 良好 |
| **Activiti** | 历史悠久 | 中小型 | 一般 |
| **ruoyi-workflow** | 国产开源、简单易用 | 快速开发 | 优秀 |
| **自研** | 完全可控 | 特殊需求 | 完美 |

## 🔄 三、流程执行引擎

### 3.1 核心服务实现

```java
@Service
@Slf4j
public class WorkflowEngine {
    
    @Autowired
    private ProcessDefinitionMapper processDefMapper;
    
    @Autowired
    private ProcessInstanceMapper instanceMapper;
    
    @Autowired
    private TaskMapper taskMapper;
    
    @Autowired
    private WorkflowVariablesService variablesService;
    
    /**
     * 启动流程实例
     */
    @Transactional
    public ProcessInstance startProcess(Long userId, String processKey, 
                                        String businessKey, Map<String, Object> variables) {
        // 1. 获取最新流程定义
        ProcessDefinition processDef = processDefMapper
            .findLatestByKey(processKey);
        if (processDef == null) {
            throw new WorkflowException("流程不存在: " + processKey);
        }
        if (processDef.getSuspensionState() == 0) {
            throw new WorkflowException("流程已挂起");
        }
        
        // 2. 创建流程实例
        ProcessInstance instance = new ProcessInstance();
        instance.setProcessKey(processKey);
        instance.setInstanceNo(generateInstanceNo());
        instance.setBusinessKey(businessKey);
        instance.setOriginatorId(userId);
        instance.setStatus("running");
        instance.setStartTime(LocalDateTime.now());
        instance.setVariables(variables);
        instanceMapper.insert(instance);
        
        // 3. 保存流程变量
        variablesService.saveVariables(instance.getId(), variables);
        
        // 4. 触发开始节点
        NodeDefinition startNode = getStartNode(processDef.getId());
        triggerNode(instance, startNode, userId);
        
        return instance;
    }
    
    /**
     * 触发节点
     */
    private void triggerNode(ProcessInstance instance, NodeDefinition node, Long userId) {
        switch (node.getNodeType()) {
            case "start":
                // 找到下一个用户任务
                List<SequenceDefinition> sequences = 
                    getOutgoingSequences(node.getId());
                for (SequenceDefinition seq : sequences) {
                    NodeDefinition nextNode = getNode(seq.getTargetNode());
                    if (nextNode.getNodeType().equals("userTask")) {
                        // 创建任务
                        createTasks(instance, nextNode);
                    }
                    triggerNode(instance, nextNode, userId);
                }
                break;
                
            case "userTask":
                createTasks(instance, node);
                break;
                
            case "exclusiveGateway":
                // 计算条件，选择一条线路
                SequenceDefinition selectedSeq = 
                    evaluateExclusiveGateway(instance, node);
                NodeDefinition nextNode = getNode(selectedSeq.getTargetNode());
                triggerNode(instance, nextNode, userId);
                break;
                
            case "parallelGateway":
                // 并行网关：触发所有出线
                List<SequenceDefinition> outSequences = 
                    getOutgoingSequences(node.getId());
                for (SequenceDefinition seq : outSequences) {
                    NodeDefinition nextNode = getNode(seq.getTargetNode());
                    triggerNode(instance, nextNode, userId);
                }
                break;
                
            case "end":
                completeProcess(instance);
                break;
        }
        
        // 更新当前节点
        instance.setCurrentNodeKey(node.getNodeKey());
        instanceMapper.updateById(instance);
    }
    
    /**
     * 创建任务
     */
    private void createTasks(ProcessInstance instance, NodeDefinition node) {
        List<Long> assigneeIds = resolveAssignees(node);
        
        for (Long assigneeId : assigneeIds) {
            Task task = new Task();
            task.setInstanceId(instance.getId());
            task.setTaskKey(node.getNodeKey());
            task.setTaskName(node.getNodeName());
            task.setTaskType("approve");
            task.setAssigneeId(assigneeId);
            task.setStatus("pending");
            task.setPriority(node.getProperties().getIntValue("priority", 50));
            task.setStartTime(LocalDateTime.now());
            taskMapper.insert(task);
            
            // 发送任务通知
            notifyService.sendTaskNotify(assigneeId, task);
        }
    }
    
    /**
     * 办理任务
     */
    @Transactional
    public void completeTask(Long taskId, Long userId, String action, 
                             String comment, Map<String, Object> variables) {
        Task task = taskMapper.selectById(taskId);
        validateTaskAssignee(task, userId);
        
        ProcessInstance instance = instanceMapper.selectById(task.getInstanceId());
        
        // 更新变量
        if (variables != null && !variables.isEmpty()) {
            variablesService.saveVariables(instance.getId(), variables);
        }
        
        // 完成任务
        task.setStatus("completed");
        task.setEndTime(LocalDateTime.now());
        task.setComment(comment);
        task.setDuration(Duration.between(task.getStartTime(), task.getEndTime()).toMillis());
        taskMapper.updateById(task);
        
        // 记录历史
        historyService.saveHistory(task, action);
        
        // 触发下一节点
        if ("approve".equals(action)) {
            NodeDefinition currentNode = getNode(task.getTaskKey());
            List<SequenceDefinition> outSequences = 
                getOutgoingSequences(currentNode.getId());
            
            for (SequenceDefinition seq : outSequences) {
                // 条件判断
                if (seq.getConditionType().equals("expression") 
                    && !evaluateCondition(instance, seq)) {
                    continue;
                }
                
                NodeDefinition nextNode = getNode(seq.getTargetNode());
                triggerNode(instance, nextNode, userId);
                break;
            }
        } else if ("reject".equals(action)) {
            // 驳回到发起人
            rejectToOriginator(instance, userId, comment);
        }
    }
    
    /**
     * 驳回
     */
    private void rejectToOriginator(ProcessInstance instance, Long userId, String comment) {
        // 驳回逻辑：创建驳回任务给发起人
        NodeDefinition rejectNode = getFirstTaskNode(instance.getProcessDefId());
        
        Task rejectTask = new Task();
        rejectTask.setInstanceId(instance.getId());
        rejectTask.setTaskKey("reject_task");
        rejectTask.setTaskName("驳回待处理");
        rejectTask.setAssigneeId(instance.getOriginatorId());
        rejectTask.setStatus("pending");
        rejectTask.setComment("驳回原因: " + comment);
        taskMapper.insert(rejectTask);
        
        notifyService.sendTaskNotify(instance.getOriginatorId(), rejectTask);
    }
}
```

### 3.2 条件表达式引擎

```java
@Service
public class ExpressionEngine {
    
    /**
     * 执行条件表达式
     */
    public boolean evaluate(ProcessInstance instance, String expression) {
        try {
            // 解析变量
            Map<String, Object> variables = getVariables(instance);
            
            // 替换变量引用
            String parsedExpr = parseExpression(expression, variables);
            
            // 使用SpEL执行表达式
            ExpressionParser parser = new SpelExpressionParser();
            return Boolean.TRUE.equals(parser.parseExpression(parsedExpr).getValue());
            
        } catch (Exception e) {
            log.error("表达式执行失败: {}", expression, e);
            return false;
        }
    }
    
    /**
     * 解析表达式，替换变量
     * 例如: ${amount > 100000} + amount=500000 => true
     */
    private String parseExpression(String expression, Map<String, Object> variables) {
        String result = expression;
        
        // 匹配 ${xxx} 格式
        Pattern pattern = Pattern.compile("\\$\\{([^}]+)\\}");
        Matcher matcher = pattern.matcher(expression);
        
        StringBuffer sb = new StringBuffer();
        while (matcher.find()) {
            String varExpr = matcher.group(1);
            Object value = evaluateVariable(varExpr, variables);
            if (value instanceof String) {
                matcher.appendReplacement(sb, "'" + value + "'");
            } else {
                matcher.appendReplacement(sb, String.valueOf(value));
            }
        }
        matcher.appendTail(sb);
        
        return sb.toString();
    }
    
    private Object evaluateVariable(String expr, Map<String, Object> variables) {
        // 支持嵌套属性: ${order.amount}
        if (expr.contains(".")) {
            String[] parts = expr.split("\\.");
            Object value = variables.get(parts[0]);
            for (int i = 1; i < parts.length && value != null; i++) {
                value = getPropertyValue(value, parts[i]);
            }
            return value;
        }
        return variables.get(expr);
    }
}
```

```java
@Service
public class WorkflowNotifyService {
    
    @Autowired
    private NotifyTemplateService notifyTemplate;
    
    /**
     * 发送任务通知
     */
    public void sendTaskNotify(Long userId, Task task) {
        User user = userService.getById(userId);
        ProcessInstance instance = instanceMapper.selectById(task.getInstanceId());
        
        // 消息通知
        Message message = Message.builder()
            .receiverId(userId)
            .title("待办任务提醒")
            .content(String.format("您有一个待审批任务：%s", instance.getTitle()))
            .bizType("workflow_task")
            .bizId(task.getId())
            .build();
        messageService.send(message);
        
        // 微信通知
        String templateCode = "workflow_task_reminder";
        Map<String, String> params = new HashMap<>();
        params.put("taskName", task.getTaskName());
        params.put("title", instance.getTitle());
        params.put("time", task.getStartTime().format(DateTimeFormatter.ofPattern("MM-dd HH:mm")));
        params.put("url", "/workflow/task/detail/" + task.getId());
        
        wechatNotifyService.send(user.getOpenId(), templateCode, params);
        
        // 短信通知（高优先级任务）
        if (task.getPriority() > 80) {
            smsService.send(user.getPhone(), "您有紧急审批任务待处理");
        }
    }
    
    /**
     * 任务超时提醒
     */
    @Scheduled(cron = "0 */5 * * * ?")  // 每5分钟
    public void sendTimeoutReminders() {
        List<Task> timeoutTasks = taskMapper.selectTimeoutTasks();
        
        for (Task task : timeoutTasks) {
            // 更新超时标记
            task.setTimeout(true);
            taskMapper.updateById(task);
            
            // 发送超时提醒
            sendTaskNotify(task.getAssigneeId(), task);
            
            // 升级处理
            if (task.getTimeoutCount() > 3) {
                escalateTask(task);
            }
        }
    }
}
```

```java
@RestController
@RequestMapping("/api/workflow")
public class WorkflowController {
    
    /**
     * 启动流程
     */
    @PostMapping("/start")
    public Result<ProcessInstanceVO> startProcess(
            @RequestHeader("X-User-Id") Long userId,

        ProcessInstance instance = workflowEngine.startProcess(
            userId, 
            dto.getProcessKey(), 
            dto.getBusinessKey(),
            dto.getVariables()
        );
        
        return Result.success(toInstanceVO(instance));
    }
    
    /**
     * 获取待办任务
     */
    @GetMapping("/tasks/todo")
    public Result<PageResult<TaskVO>> getTodoTasks(
            @RequestHeader("X-User-Id") Long userId,
            @RequestParam(defaultValue = "1") Integer page,
            @RequestParam(defaultValue = "10") Integer size) {
        
        Page<Task> tasks = workflowEngine.getTodoTasks(userId, page, size);
        return Result.success(PageResult.of(tasks, this::toTaskVO));
    }
    
    /**
     * 完成任务
     */
    @PostMapping("/tasks/{taskId}/complete")
    public Result<Void> completeTask(
            @RequestHeader("X-User-Id") Long userId,
            @PathVariable Long taskId,
            @RequestBody CompleteTaskDTO dto) {
        
        workflowEngine.completeTask(
            taskId, 
            userId, 
            dto.getAction(),  // approve/reject/transfer
            dto.getComment(),
            dto.getVariables()
        );
        
        return Result.success();
    }
    
    /**
     * 获取流程图
     */
    @GetMapping("/instances/{instanceId}/diagram")
    public Result<String> getProcessDiagram(@PathVariable Long instanceId) {
        String svg = workflowEngine.getProcessDiagram(instanceId);
        return Result.success(svg);
    }
    
    /**
     * 获取流程历史
     */
    @GetMapping("/instances/{instanceId}/history")
    public Result<List<HistoryTaskVO>> getProcessHistory(@PathVariable Long instanceId) {
        List<HistoryTask> histories = workflowEngine.getHistory(instanceId);
        return Result.success(histories.stream().map(this::toHistoryVO).collect(toList()));
    }
}
```

## ⚠️ 七、常见问题与解决方案

| 问题 | 解决方案 |
|------|----------|
| 审批人离职/调岗 | 动态获取审批人+委托机制 |
| 流程变更后实例如何处理 | 历史实例走原流程，新实例用新流程 |
| 会签/或签逻辑复杂 | 并行网关+聚合表达式 |
| 流程性能瓶颈 | 异步执行+数据库索引优化 |
| 流程数据与业务数据一致性 | TCC事务+补偿机制 |
