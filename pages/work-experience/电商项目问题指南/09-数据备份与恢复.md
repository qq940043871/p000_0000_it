# 09 - 数据备份与恢复

> 如何保证数据不丢失、灾难可恢复

---

## 📖 概述

数据是电商最核心的资产。本章介绍多层次的数据保护策略。

---

## 🗄️ 一、备份策略

### 1.1 备份类型

| 类型 | 频率 | 恢复时间 | 存储成本 |
|------|------|----------|----------|
| **全量备份** | 每周/天 | 长 | 高 |
| **增量备份** | 每小时 | 中 | 中 |
| **日志备份** | 实时 | 短 | 低 |

### 1.2 备份架构

```
┌─────────────────────────────────────────────┐
│              数据备份架构                        │
│                                             │
│  ┌─────────┐     实时      ┌─────────┐     │
│  │  主库   │ ──────────► │ Binlog │     │
│  │ MySQL   │              │ Server │     │
│  └─────────┘              └────┬────┘     │
│        │                       │           │
│        │ 全量/增量             │ 增量同步   │
│        ▼                       ▼           │
│  ┌─────────┐              ┌─────────┐     │
│  │  备份库  │ ◄─────────│ 备库   │     │
│  │ (本地)  │              │ (远程) │     │
│  └─────────┘              └─────────┘     │
└─────────────────────────────────────────────┘
```

---

## 🔄 二、主从同步

### 2.1 异步复制

```ini
# MySQL 配置
[mysqld]
log-bin = mysql-bin
server-id = 1
binlog_format = ROW
sync_binlog = 1
```

## 💾 三、大厂实践

| 平台 | 备份策略 | RPO | RTO |
|------|----------|-----|-----|
| 阿里 | 单元化+三地备份 | < 1秒 | < 5分钟 |
| 京东 | MGR+异地同步 | < 1秒 | < 10分钟 |
| 携程 | 双活+日志回放 | < 1分钟 | < 30分钟 |
