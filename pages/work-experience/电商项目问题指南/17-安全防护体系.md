# 17 - 安全防护体系

> 如何保障系统安全

---

## 📖 概述

本章介绍XSS、SQL注入、CSRF防护等安全措施。

---

## 🛡️ 一、常见攻击

| 攻击 | 说明 | 防护 |
|------|------|------|
| **XSS** | 跨站脚本攻击 | 转义/过滤 |
| **SQL注入** | 恶意SQL执行 | 预编译 |
| **CSRF** | 跨站请求伪造 | Token |
| **DDoS** | 分布式拒绝服务 | 限流/清洗 |

---

## 🔒 二、接口幂等

```java
// 接口幂等注解
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface Idempotent {
    long expireSeconds() default 60;
}

// 幂等拦截器
public class IdempotentInterceptor {
    public boolean preHandle(String key) {
        return redisTemplate.setIfAbsent(key, "1", 60, TimeUnit.SECONDS);
    }
}
```
