## 📝 一、工单系统

### 1.1 工单状态机

```
[待处理] ──► [处理中] ──► [待确认] ──► [已解决]
    │            │            │
    ▼            ▼            ▼
[已退回]     [升级处理]   [已关闭]
```

### 1.2 工单表设计

```sql
CREATE TABLE `t_work_order` (
  `id` BIGINT PRIMARY KEY,
  `order_id` BIGINT,
  `user_id` BIGINT,
  `type` TINYINT,           -- 1=退款 2=退货退款 3=投诉 4=其他
  `status` TINYINT,         -- 1=待处理 2=处理中 3=待确认 4=已解决
  `reason` VARCHAR(500),
  `images` VARCHAR(2000),    -- 图片凭证
  `created_at` DATETIME,
  `updated_at` DATETIME
);
```

## 🔄 二、售后处理

### 2.1 退款流程

```java
@Service
public class RefundService {
    
    @Transactional
    public void processRefund(Long orderId, RefundDTO dto) {
        Order order = orderService.getById(orderId);
        
        // 1. 校验退款金额
        if (dto.getAmount() > order.getPayAmount()) {
            throw new BizException("退款金额超出");
        }
        
        // 2. 更新订单状态
        order.setStatus(OrderStatus.REFUNDING);
        orderDao.updateById(order);
        
        // 3. 原路退回
        refundPayment(order.getPayChannel(), order.getTradeNo(), dto.getAmount());
        
        // 4. 更新状态
        order.setStatus(OrderStatus.REFUNDED);
        orderDao.updateById(order);
        
        // 5. 发送退款消息
        mqProducer.send("order-refund", order);
    }
}
```

## ⭐ 三、评价体系

### 3.1 评价数据模型

```sql
CREATE TABLE `t_review` (
  `id` BIGINT PRIMARY KEY,
  `order_id` BIGINT,
  `product_id` BIGINT,
  `user_id` BIGINT,
  `rating` TINYINT,           -- 1-5星
  `content` VARCHAR(500),
  `images` VARCHAR(2000),
  `reply` VARCHAR(500),         -- 商家回复

  `created_at` DATETIME

CREATE TABLE `t_product_stats` (
  `product_id` BIGINT PRIMARY KEY,
  `rating_avg` DECIMAL(3,2),   -- 平均评分

  `rating_1` INT,             -- 1星数
  `rating_2` INT,
  `rating_3` INT,
  `rating_4` INT,
  `rating_5` INT,
  `updated_at` DATETIME
);
```
